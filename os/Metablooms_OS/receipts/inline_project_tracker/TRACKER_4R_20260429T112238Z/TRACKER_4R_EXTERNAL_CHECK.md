# TRACKER-4R External Check

- W3C WCAG Reflow guidance: content should reflow within narrow viewports without loss of information/functionality or two-dimensional scrolling.
- ARIA/progress guidance: do not imply determinate progress when progress is unknown; keep explicit status text.
- User visual evidence: mobile screenshots of TRACKER-4 proved box-drawing/right-border layout became unreadable after ChatGPT mobile wrapping.

Decision: compact stacked tracker with a small determinate bar only when stage denominator is known.
